# Marwa's Diary Book 💗

A personal diary and planner web app with a soft pink design.

## Features
- Login page
- Weekly planner
- Monthly calendar
- Daily planner by 30-minute slots
- Tasks with checkboxes
- Edit and delete tasks
- Upload and delete images
- Image preview
- Mood tracker
- Reminders
- Dark mode
- Export day as PDF
- Mobile responsive design
- PWA support

## Technologies
- HTML
- CSS
- JavaScript
- LocalStorage
- Firebase Messaging
- GitHub Pages
